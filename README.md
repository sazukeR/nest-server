1. Clone the project.

2. ```npm install```

3. Clone file ```.env.template``` and rename to ```.env```

4. Change the environment variables.

5. Up database.
  ```
  docker-compose up -d
  ```
6. Start ```npm run start:dev```

7. Run seed.
 ```
 http://localhost:3000/api/seed
 ```
